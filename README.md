## 🔬 Theoretical Framework

The project is built around an informational field theory where agent interactions warp the underlying state space. This dynamics is modeled through geometric field equations analogous to the **Einstein Field Equations** in modern physics:

$$\mathcal{R}_{\mu\nu} - \frac{1}{2} \mathcal{R} g_{\mu\nu} + \Lambda g_{\mu\nu} = \kappa T_{\mu\nu}$$

Where:
* **$g_{\mu\nu}$ (Induced Metric):** The computational metric tensor derived dynamically from state-dependent interaction functions $\gamma(t, I_S)$.
* **$\mathcal{R}$ (Ricci Scalar):** Spatial curvature representing the distortion and density stress of the underlying Interaction Space.
* **$T_{\mu\nu}$ (Reinforcement Source Tensor):** The energy-momentum analogue calculated from the moments of reinforced agent response distribution densities.
* **$\kappa, \Lambda$:** The coupling coefficient and the operational cosmological constant respectively.

### Fokker-Planck Response Dynamics

The response density $P(t, I_S)$ evolves through a specialized Advection-Diffusion equation governed by custom reinforcement structures:

$$\frac{\partial P}{\partial t} = -\partial_i (R^i P) + \frac{1}{2} \partial_i \partial_j (D^{ij} P) + \Sigma$$

* **$R^i$ (Reinforcement Drift):** Target-driven advection steering agents toward appetitive milestones and away from aversive states.
* **$D^{ij}$ (Diffusion Tensor):** Variable dispersion of agent behaviors dictated by operational reinforcement schedules (e.g., Fixed-Ratio, Variable-Interval).
* **$\Sigma$ (Source/Sink term):** Rate components determining behavior acquisition and behavioral extinction dynamics.

## 💻 Numerical Engine (`2d+3d.py`)

The `2d+3d.py` script computes these coupled field-agent differential systems numerically and provides an interactive, dual-pane layout:

1. **Left Panel (2D View):** Visualizes the active field residual $R^{space} - (\kappa T_{00} + \Lambda)$ via sharp contour mapping, coupled with reinforcement attraction vector fields (Quiver Plot) and moment trajectories.
2. **Right Panel (3D View):** Renders the 3D interaction manifold geometry generated by the induced Ricci scalar alongside the active tracking of the agent's expected state $E[I_S]$ relative to the curvature.

### Technical Breakdown:
* **Finite Volume Advection:** Leverages an upwind finite volume method to provide high numerical stability and strictly maintain mass conservation across no-flux boundaries.
* **Numerical Differential Geometry Engine:** Computes Christoffel symbols and Ricci curvature metrics directly from empirical spatial grids using finite difference approximation.
* **Fully Coupled Feedback Loop:** Interdependently updates the geometric field metrics and the agent's probability distributions at each timestep ($dt$).

## 🛠️ Installation & Usage

The engine is developed in Python 3 and relies on standard scientific libraries. Install them via:

```bash
pip install numpy matplotlib scipy
```

You can initiate the interactive numerical simulation environment with the following command:

python 2d+3d.py
